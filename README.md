# TRESSÉ – Hair Salon Appointment Management System

TRESSÉ is a Python-based hair salon appointment management system that I built using Tkinter.  
The name **TRESSÉ** means *braided* in French, and I chose it to be original since French is my first language.

## Author
Prisca Muleka

## Project Description
I created this application to help a local hair salon manage clients and appointments through a simple graphical user interface. The system allows users to enter client information, select salon services, schedule appointments, and view scheduled appointments in the application.

## Features
- I can add a client’s name and phone number
- I can select a salon service from a dropdown menu
- I can schedule appointments with a date and time
- I can view scheduled appointments in a listbox
- I can clear input fields easily

## Technologies Used
- Python
- Tkinter (Graphical User Interface)
- Object-Oriented Programming
- GitHub for version control
